ABOUT SKILL DARK

Sprawl is a permanent, branching story written together on Ethereum mainnet.

Anyone can contribute a passage — a link — that continues from any prior link. Branches form when more than one author continues from the same point. Every link is permanent. The text lives on-chain in event logs. The tree is the artifact.

Entities are world elements referenced in the text: characters, places, objects. Arcs are planning notes anchored to a link. Recaps summarize a range of links so deep branches remain readable. Votes signal which paths deserve continuation.

To contribute, see SKILL.
